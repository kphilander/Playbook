# All Employees

**The universal foundation. Every staff member completes this before any role-specific training.**

> **Operator note**: This program establishes the baseline understanding of {{PROGRAM_NAME}} that all other programs build on. Customize examples and integration points in `_brand.yml` before deploying. All employees — from floor staff to executives — complete this program.

## What you'll be able to do after this program

- Explain what {{PROGRAM_NAME}} is (and what it isn't) in one sentence
- Name the two brand pillars and their taglines
- Define the Knowledge Gap and how it relates to the Engagement Gap
- State the four premises of the informed choice model
- Distinguish between Tier 1 and Tier 2 content
- Recognize and avoid stigmatizing language in everyday conversation
- Know when to apply Tier 1 voice and when to shift to Tier 2
