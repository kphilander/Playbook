# Marcomm

**Deep training for marketing and creative teams who build and deploy {{PROGRAM_NAME}} content.**

> **Operator note**: This program has two tracks. The **Build track** (included below) trains product marketing and creative teams to create content using the {{PROGRAM_NAME}} brand system. The **Deploy track** (for operators implementing the system) is planned for a future release.

---

## Program summary

| | |
|---|---|
| **Audience** | Marketing, creative, content teams |
| **Modules** | 3 |
| **Duration** | ~180 minutes (3 hours) |
| **Prerequisites** | All Employees (both modules, passed) |
| **Passing score** | 80% on Module 00 quiz; Modules 01-02 use reflection-based assessment |
| **Certification** | Brand & Content Certificate |
| **Recertification** | Every 12 months |

## Modules — Build Track

| # | Module | Duration | What it covers |
|---|--------|----------|---------------|
| 00 | [Brand System Deep Dive](00-brand-system-deep-dive.md) | ~60 min | Full brand architecture, visual identity, voice/tone, cultural adaptation |
| 01 | [Content Creation Workshop](01-content-creation-workshop.md) | ~60 min | Core messages, taglines, CTAs, myth-busting, quiz creation |
| 02 | [Campaign Planning & Measurement](02-campaign-planning-and-measurement.md) | ~60 min | 7 campaign briefs, channel specs, player segments, measurement |

## Deploy Track (Planned)

The following modules are planned for future release:

| # | Module | Description |
|---|--------|-------------|
| D-00 | Operator Onboarding | How to fork, configure `_brand.yml`, and set up the brand system |
| D-01 | White-Label Configuration | Customizing color, typography, messaging, and helpline settings |
| D-02 | Market Launch Playbook | Go-live checklist, stakeholder alignment, measurement setup |

## What you'll be able to do after this program

- Navigate the full {{PROGRAM_NAME}} brand system: `_brand.yml`, brand book, visual identity, messaging library
- Write on-brand content in any tone register using the 74-message library
- Build campaigns using the 7 ready-to-run briefs and collateral template system
- Adapt content across cultural profiles using the 5-dimension framework
- Measure content effectiveness using the player testing protocol
- Review and approve content against the brand rubric
