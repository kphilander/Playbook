## Modules

| # | Module | Duration | What it covers |
|---|--------|----------|---------------|
| 00 | [Player Tools Walkthrough](00-player-tools-walkthrough.md) | ~20 min | Every player-facing tool — mechanics, timing, reversibility, one-sentence explanations |

| 02 | [Conversations in Practice](02-conversations-in-practice.md) | ~15 min | Tone principles for live conversations, three scenario walkthroughs, escalation criteria |

## What you'll be able to do after this program

- Explain every player-facing tool in one sentence — what it does, how it works, whether it's reversible
- Guide a player through setting up deposit limits, session reminders, cool-off, or self-exclusion
- Use the full 35-term stigma-free vocabulary in live player interactions
- Distinguish Tier 1 and Tier 2 language choices in real-time conversation
- Handle low-risk player conversations (deposit limits, odds questions, account pauses) using {{PROGRAM_NAME}} tone principles
- Recognize when a conversation needs escalation and hand off appropriately
