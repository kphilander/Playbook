## Print specs

| Property | Value |
|---|---|
| **Flat size** | A4 (297 x 210mm) or Letter (11 x 8.5") |
| **Folded size** | 99 x 210mm or 3.67 x 8.5" |
| **Fold type** | Letter fold (tri-fold, right panel folds in first) |
| **Bleed** | 3mm / 0.125" on all sides |
| **Safe zone** | 5mm / 0.25" inside trim on all edges |
| **Color mode** | CMYK |
| **Resolution** | 300 DPI minimum |
| **Paper** | 170gsm coated silk or equivalent |
| **Coating** | Soft-touch matte laminate on cover panel |
