# Changelog

All notable changes to Playbook are documented here.

Format follows [Semantic Versioning](https://semver.org/):
- **MAJOR** = breaking changes to brand identity or architecture
- **MINOR** = new content (chapters, jurisdictions, collateral)
- **PATCH** = corrections, regulatory updates, clarifications

## [0.1.0] - 2025-02-20

### Added
- `_brand.yml` white-label configuration engine with four brand pillars (Sharp, Choice, Open, Social), color palette, typography, helplines, tagline system, and tone configuration
- Brand book: 10 chapters covering introduction, brand foundation, personality, visual identity, voice and tone, messaging framework, accessibility, application guidelines, governance, and glossary
- Project infrastructure: README, CONTRIBUTING guidelines, CC0 license
- Two-tier brand architecture: Tier 1 (entertainment literacy) and Tier 2 (support/crisis, planned)
- Reno Model positioning: informed choice framing, marketing-oriented voice, confident myth-buster tone
