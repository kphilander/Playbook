# Changelog

All notable changes to the Compass brand guidelines system are documented here.

This project follows [Semantic Versioning](https://semver.org/): MAJOR.MINOR.PATCH where:
- **MAJOR** = breaking changes to brand identity or system architecture
- **MINOR** = new content (chapters, jurisdiction modules, collateral templates)
- **PATCH** = corrections, clarifications, regulatory updates

## [0.1.0] - 2025-02-19

### Added
- **Brand configuration**: `_brand.yml` white-label configuration engine with Compass defaults for colors, typography, helplines, messaging, and legal requirements
- **Brand book foundation**: 10 chapter brand guidelines covering introduction, brand foundation, personality, visual identity, voice and tone, messaging framework, accessibility, application guidelines, governance, and glossary
- **Project infrastructure**: README with quick-start guide, CONTRIBUTING guidelines, CC0 license
- **Multi-jurisdiction helplines**: Default helpline configurations for US, UK, Australia, Canada, and EU
- **Modern wellness positioning**: Brand identity anchored in engagement-first, non-stigmatizing, player-empowering principles
