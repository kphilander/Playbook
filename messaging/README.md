# Messaging

The complete messaging content library for Playbook. Every piece of copy — taglines, CTAs, core messages, myth-busting content, and tone examples — lives here. Use these documents as your primary reference when creating any Playbook content.

## Documents

| Document | Description |
|----------|-------------|
| [Core Messages](core-messages.md) | 74 jurisdiction-agnostic messages organized by touchpoint, tagged with pillar, tone register, tier, and character count |
| [Tagline System](tagline-system.md) | Tagline hierarchy, rotation guidance, and framework for writing new taglines |
| [Calls to Action](calls-to-action.md) | CTA library organized by player action — label text, character limits, context, and tone register |
| [Myth-Busting](myth-busting.md) | Common gambling myths debunked in three formats: social card copy, article explainer, and quiz question |
| [Campaigns](campaigns.md) | Six ready-to-run campaign briefs with day-by-day schedules, channel specs, and template references |
| [Stigma-Free Language](stigma-free-language.md) | 35-term language guide with preferred alternatives, rationale, and examples in context |
| [Tone Examples](tone-examples.md) | Before/after pairs showing the Playbook voice transformation across scenarios |

## How to use this

- **Writing new content?** Start with [Core Messages](core-messages.md) for ready-to-use copy, then check [Tone Examples](tone-examples.md) for voice alignment.
- **Planning a campaign?** [Campaigns](campaigns.md) has six complete briefs. Mix with taglines from [Tagline System](tagline-system.md) and CTAs from [Calls to Action](calls-to-action.md).
- **Targeting a specific audience?** The [player-segments/](player-segments/) profiles have tailored messaging for each group.
