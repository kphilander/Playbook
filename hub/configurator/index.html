Playbook content is designed to support responsible gambling programs, not replace regulatory compliance. Requirements for mandatory disclosures, helpline display, and player communications vary by jurisdiction. Operators should review all content with their compliance team before deployment.

What's Next?

  1. Fork the Playbook repo on GitHub
  2. Replace _brand.yml with the file you just downloaded
  3. Place your logo file in visual-identity/logo/operator/
  4. Run npm install && npm run build to regenerate all assets
  5. Run cd collateral/render && npm install && node render-cards.mjs to render all 45 templates as PNGs
